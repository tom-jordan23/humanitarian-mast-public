# docs/ — Public (sanitized) engineering documentation

All docs here are **public-safe**: sanitized, generalized, or synthetic. No real address,
coordinates, parcel geometry, or identifiable imagery. Anything location-bearing stays in the
private repo until it passes the pre-publication review gate (`/publication-review`).

Every doc should distinguish: **settled decisions · working assumptions · open questions · deferred
features · safety requirements · field-verification tasks.**
