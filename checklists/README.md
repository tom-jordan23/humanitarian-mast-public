# checklists/ — Field-usable checklists

Written to be used **outdoors during construction**. Planned checklists:

| File | When |
|------|------|
| `pre-dig.md` | Before any trenching/anchor excavation — **utility locate first** |
| `mast-raise.md` | Mast assembly, guy install & tensioning, raise |
| `grounding-inspection.md` | Grounding/bonding verification |
| `frame-relocation-bonding.md` | Before re-energizing a relocated power frame (experimentation frame, or an isolation frame moved off its normal mast-base bond) — bonding-path/continuity/bulkhead-seal/sign-off gate |
| `rf-test.md` | Basic continuity and RF checks |
| `field-box-test.md` | Field enclosure / Pi / power-cycle recovery |
| `publication-review.md` | Privacy gate before private → public |
| `storm-shutdown.md` | Storm / shutdown procedure |

Safety, permitting, grounding, and structural concerns must be called out clearly, including any
work requiring a tower professional, electrician, or local authority. Use the
`grounding-safety-inspector` and `field-docs-writer` sub-agents.
