# lighting-controller/ — Mast / tree / site lighting control

Low-power, DC-native, field-replicable lighting control for the mast/site. Part of the first build.

Planned files: `wiring.md`, `controller-config.md`, `test-plan.md`.

Acceptance: lighting controller works **locally and remotely**, runs in a low-power DC mode, and
recovers after power loss. Use the `field-station-engineer` sub-agent.
